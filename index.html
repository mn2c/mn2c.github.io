About

This website is started and maintained by me alone. My name is Martinn, and I'm a data analytics professional with a day job outside the real estate industry, you can view my profile here. I began this site as a weekend project with the goal of learning new skills through hands-on experience and contributing to the public with my data skills.

I intentionally keep the site ad-free to ensure a pleasant user experience, although I do incur some costs for domain and cloud infrastructure.

If you're planning a property purchase, you may find the Property Purchase Planner Tool section useful. Unlike conventional loan calculators, this Singapore-specific property purchase planning tool allows you to effortlessly experiment with various input combinations (such as price, interest, tenure, cash/CPF, etc.) using sliders. It calculates everything in real-time, enabling you to swiftly model different financial scenarios with lightning speed!
